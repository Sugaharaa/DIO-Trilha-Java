# Desafio Controle de Fluxo

Projeto desenvolvido como desafio do módulo **Controle de Fluxo** da trilha Java Básico da DIO.

O sistema recebe dois números inteiros via terminal e realiza uma contagem baseada na diferença entre eles.

---

## Regras do sistema

- O usuário informa dois números inteiros:
  - Primeiro parâmetro
  - Segundo parâmetro
- O programa calcula a diferença entre os dois valores.
- A quantidade de interações do `for` será igual a essa diferença.
